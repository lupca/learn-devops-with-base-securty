Roadmap 30 ngày: DevSecOps Thực Chiến

Chuyển đổi kỹ năng từ Security Engineer sang DevSecOps. Xây dựng hệ thống an toàn, tự động và chuẩn mực.

Giai đoạn 1: Linux & Networking

Xây dựng nền tảng OS và Mạng cơ bản.

Góc nhìn Security:
- Phân quyền file (chmod/chown) chính là quản trị ACL/RBAC.
- Terminal là công cụ trinh sát. Mọi cấu hình sai (misconfig) đều nằm ở file text.

Giai đoạn 1: Linux & Networking

Xây dựng nền tảng OS và Mạng cơ bản.

  • Linux Admin: Thao tác Bash shell, xử lý log, quản lý Process.
  • Network & SSH: Quản lý IP, Port. Setup SSH Key (Asymmetric Auth).
  • Virtualization: Triển khai VM tự động để test (Multipass/Vagrant).

Giai đoạn 2: Containerization

Đóng gói ứng dụng với Docker & Kubernetes.

  • Docker Core: Đóng gói ứng dụng. Image scanning để phát hiện CVE.
  • Docker Compose: Điều phối multi-container, cấu hình network cô lập.
  • K3s/K8s Basics: Quản lý Pod, Deployment, Service, và NetworkPolicy.

Giai đoạn 2: Containerization

Đóng gói ứng dụng với Docker & Kubernetes.

Góc nhìn Security:
- Container chính là Sandbox isolation. Giới hạn tài nguyên và quyền hạn giúp giảm thiểu thiệt hại (blast radius) khi ứng dụng bị chiếm quyền.

Giai đoạn 3: Infrastructure as Code

Tự động hóa hạ tầng bằng Terraform.

Góc nhìn Security:
- File State (.tfstate) lưu trữ toàn bộ secrets của hạ tầng. Terraform plan giúp phát hiện Drift (sự thay đổi trái phép) trước khi apply.

Giai đoạn 3: Infrastructure as Code

Tự động hóa hạ tầng bằng Terraform.

  • Terraform Basics: Viết code HCL để deploy VM/Cloud Resources.
  • State Management: Quản lý tfstate an toàn, phân tích thay đổi hạ tầng.
  • Provisioning: Tự động chạy script hardening bằng Cloud-init ngay khi boot.

Giai đoạn 4: GitOps & Automation

Vận hành tự động và bảo mật CI/CD.

  • ArgoCD (GitOps): Git là Single Source of Truth. Thay đổi đều phải qua Pull Request.
  • Secret Management: Tích hợp HashiCorp Vault. Không hardcode credentials.
  • Final Project: Terraform tạo VM -> Hardened OS -> K3s/Vault/ArgoCD -> Deploy App.

Giai đoạn 4: GitOps & Automation

Vận hành tự động và bảo mật CI/CD.

Góc nhìn Security:
- GitOps mang lại Continuous Compliance. Nếu có ai đó tự ý sửa config trên server, ArgoCD sẽ ngay lập tức ghi đè lại bằng bản chuẩn trên Git (như chức năng auto-remediation).
